# Formula1_Analysis
Competitive Team Analysis
This project provides a comprehensive analysis of Formula 1 race data, focusing on time series analysis and visualization techniques. Utilizing the extensive dataset available on Kaggle, this project aims to uncover insights into constructor and driver performance, trends in lap times, and the evolution of pit stop strategies across F1 seasons from 1950 to 2025.
